1. Database Migrations
Run the following command in your Replit shell to update your Prisma schema and generate your new client models:

Bash
npx prisma migrate dev --name expand_scriptures_and_playlists
2. Seeding Your New Content
To quickly test the new database setup with multi-tradition structures, create or add this snippet to your prisma/seed.ts file to populate the Tao Te Ching:

TypeScript
import { PrismaClient } from '@prisma/client';
const prisma = new PrismaClient();

async function main() {
  const taoism = await prisma.scripture.create({
    data: {
      title: "Tao Te Ching",
      tradition: "Taoism",
      description: "The fundamental text of Taoist philosophy by Lao Tzu.",
      sections: {
        create: [
          {
            title: "Chapter 1",
            sequenceOrder: 1,
            verses: {
              create: [
                {
                  verseNumber: "1.1",
                  originalText: "道可道，非常道。名可名，非常名。",
                  translation: "The Tao that can be told is not the eternal Tao. The name that can be named is not the eternal Name.",
                  commentary: "Linguistic note: 'Tao' translates literally to 'The Way' or path."
                }
              ]
            }
          }
        ]
      }
    }
  });
  console.log("Seeded new spiritual traditions successfully.");
}

main().catch(e => console.error(e));
